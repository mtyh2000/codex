# 项目 Skill 说明

这个目录用于存放跟 `智慧农业` 项目一起同步的 Codex Skill 和相关工作流资产。

当前包含：

- `julong-project-copilot`
  面向印尼聚龙集团棕榈园智慧农业项目的长期协作 Skill

## 这个 Skill 的作用

安装后，Codex 更容易按这个项目的固定节奏继续工作，包括：

- 按 `00_总控主线 + 01~05` 板块分支推进
- 先读 `AGENTS.md / README.md / 更新日志 / 待办清单`
- 把零散讨论回收到主文档、对话归档、更新日志、待办清单
- 沿用中文、直接、长期协作的项目工作方式

## 双机使用方式

如果项目仓库已经同步到另一台电脑：

1. 先更新项目仓库
2. 再运行项目根目录里的安装脚本

脚本位置：

- Mac：`安装项目技能_mac.command`
- Windows：`安装项目技能_windows.bat`

安装完成后，如果 Codex 正在运行，建议重启一次。
