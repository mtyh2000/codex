# 03 手持终端与现场测试

本板块存放终端长期选型和真实现场测试相关资料。

## 已归档资料

- [006_手持终端长期选型建议.txt](006_手持终端长期选型建议.txt)

## 放在这里的原因

这份资料不是简单参数对比，而是在回答：

- 如果项目想做 5 到 8 年的平台化使用，终端该怎么选
- 试点期该怎么测，而不是纸面拍板
- 长期平台设备和预算型试点设备应该怎么分层

## 当前可提炼的结论

- 长期平台优先级高于单次采购价
- 正式长期平台可优先看 Zebra / Honeywell 一类企业级设备
- 试点阶段不宜一次性大批量采购，应先做 2 到 3 款设备实测
