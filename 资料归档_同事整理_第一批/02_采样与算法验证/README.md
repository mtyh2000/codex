## 放在这里的原因

这两份资料分别回答了两个核心问题：

- 图片压缩到什么程度还不至于明显伤害模型判断
- 外部数据能帮多少忙，本地样本最低还需要多少

## 当前可提炼的结论

- 100KB 可用于普通预览或初步复核，但不适合作为重点样本唯一留存格式
- 刺猬果、低置信度、争议样本应保留更高质量图片
- 公开数据和购买样本适合冷启动，但不能完全代替印尼现场样本
- 更合理的采样策略是“少量高价值样本”，不是全员大量拍照
