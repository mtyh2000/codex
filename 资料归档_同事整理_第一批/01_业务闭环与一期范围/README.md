## 放在这里的原因

这份资料本质上不是单纯的设备讨论，而是在帮助我们划清一期边界：

- 一期主线应该先以 PAD 跑通业务闭环
- 工业摄像头更适合第二阶段在果台或固定质检工位试点
- 不建议一开始就把“全园区固定相机替代人工”当主方案

这份材料适合在讨论“一期做什么、不做什么”时引用。
