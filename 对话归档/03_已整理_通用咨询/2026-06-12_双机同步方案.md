# 双机同步方案

## 背景

希望把当前 Windows 上整理的项目资料同步到 MacBook Air，并形成稳定的双机协作方式。

## 结论

当前项目已经完成：

- 本地 Git 仓库初始化
- 远程 GitHub 仓库绑定
- 首次推送完成
- README、更新日志、待办清单已同步

当前 GitHub 仓库：

- `https://github.com/mtyh2000/codex.git`

默认分支：

- `main`

## MacBook Air 上的操作

首次拉取：

```bash
git clone https://github.com/mtyh2000/codex.git
cd codex
```

开始工作前：

```bash
git pull
```

完成修改后：

```bash
git add .
git commit -m "写这次修改说明"
git push
```

## 协作提醒

- 两台电脑不要同时改同一份文件后直接各自推送
- 每次开始工作前先 `git pull`
- `doc_extract/` 这类中间提取目录不会同步
