# 智慧农业项目

## 项目定位

本项目服务于印尼聚龙集团棕榈园智能化改造，当前重点聚焦一期业务闭环：

- 园区收果
- PAD / 手持终端拍照识别
- 端侧离线识别与弱网补传
- 运输调度与工厂验收比对
- 数据沉淀、复核、统计与溯源

当前不把病虫害、灌溉、遥感、机器人等长期能力作为一期主目标。

## 当前工作重点

现阶段主要在做项目评估和方案收敛，重点包括：

- 明确一期业务边界
- 组织现场采样与标注规则
- 验证手持终端真实可用性
- 梳理平台字段、复核流程和系统对接
- 准备商务测算和实施路径
