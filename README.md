# 智慧农业项目

## 项目定位

本项目服务于印尼聚龙集团棕榈园智能化改造，当前重点聚焦一期业务闭环：

- 园区收果
- PAD / 手持终端拍照识别
- 端侧离线识别与弱网补传
- 运输调度与工厂验收比对
- 数据沉淀、复核、统计与溯源

当前不把病虫害、灌溉、遥感、机器人等长期能力作为一期主目标。

## 当前工作重点

现阶段主要在做项目评估和方案收敛，重点包括：

- 明确一期业务边界
- 组织现场采样与标注规则
- 验证手持终端真实可用性
- 梳理平台字段、复核流程和系统对接
- 准备商务测算和实施路径

## 同步方式

当前仓库已经接入 GitHub：

- 仓库地址：`https://github.com/mtyh2000/codex.git`
- 默认分支：`main`

### 推荐方式

优先使用根目录里的同步脚本：

- Mac：`一键同步_mac.command`
- Windows：`一键同步_windows.bat`

详细说明见 [双机同步说明.md](双机同步说明.md)。

### 手动方式

如果临时不用脚本，也可以继续手动执行：

```bash
git pull
git add .
git commit -m "写这次修改说明"
git push
```

## 协作提醒

- 尽量不要两台电脑同时修改同一份文件后再各自直接推送
- 开始工作前先 `git pull`
- `doc_extract/` 这类中间提取目录不会同步
- 正式对客户输出前，尽量先把结论收敛到主文档，再生成 PPT 或汇报材料
